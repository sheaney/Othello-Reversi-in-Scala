# Othello-Reversi-in-Scala

You are welcome to fork this project and modify it.

## Requirements

> java 1.6 or higher

> sbt 0.12.x

## Building

Change into main directory and run:

0. Install [sbt 0.12.2](http://www.scala-sbt.org/release/docs/Getting-Started/Setup.html)
1. ```sbt update```
2. ```sbt compile```
3. ```sbt run```
