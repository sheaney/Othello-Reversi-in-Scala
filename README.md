Othello-Reversi-in-Scala
========================

Othello/Reversi Game developed in Scala

You are welcome to change/modify this project

Requirements
scala 2.9.x
java 1.6 or higher
sbt 0.10 or higher

Installation
Change into main directory and run
> sbt compile
> sbt run
